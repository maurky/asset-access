Asset Accessibility Demo

Un widget di accessibilità leggero e conforme WCAG 2.1 AA, installabile con un singolo script. Prova il pulsante in basso a destra.

Come Installarlo

Bastano due tag <script> prima della chiusura del </body>. Il primo carica il widget, il secondo imposta la configurazione.

<!-- 1. Carica lo script -->
<script src="asset-accessibility.js"></script>

<!-- 2. Configura (contactEmail e/o contactPhone OBBLIGATORI) -->
<script>
  var AssetAccessibilityConfig = {
    contactEmail: 'accessibilita@esempio.it',
    contactPhone: '+39 02 1234567',
    position: 'bottom-right',
    buttonColor: '#1a56db',
    callback: function(e) { console.log(e); }
  };
</script>

Opzioni di Configurazione

Parametro Default Descrizione
contactEmail — ⚠️ Obbligatorio* — Email per segnalazioni sull'accessibilità
contactPhone — ⚠️ Obbligatorio* — Telefono per segnalazioni sull'accessibilità
* Almeno uno tra contactEmail e contactPhone deve essere specificato, altrimenti il widget non si avvia.
position 'bottom-right' Posizione del pulsante: bottom-right, bottom-left, top-right, top-left
buttonColor '#1a56db' Colore principale del bottone e degli accenti
buttonSize 56 Dimensione in px del bottone trigger
buttonIcon 'default' Icona SVG custom (stringa) oppure 'default'
lang auto-detect Lingua iniziale: 'it', 'en', 'fr', 'de', 'es'. Rilevata dal browser se omessa.
callback null Funzione invocata a ogni interazione: function(event) { }
statementText auto-generato Oggetto con chiavi 'it' e 'en' contenenti HTML della dichiarazione. Se omesso, viene generato automaticamente dai contatti.
zIndex 999999 z-index del widget

Funzionalità Incluse

Il widget copre le principali regolazioni richieste per l'accessibilità web, tutte applicabili in tempo reale:

👤 Profili

Configurazioni predefinite con un click: Accessibilità, Ipovedente, ADHD, Disabilità Cognitiva, Dislessia.

📝 Contenuti

Dimensione testo, carattere leggibile, allineamento, altezza linea, spaziatura caratteri.

🎨 Colori

Temi a contrasto scuro, chiaro ed elevato. Modalità monocromatica. Saturazione alta e bassa.

⚙️ Altro

Nascondi immagini, cursore grande, ferma animazioni. Cambio lingua (IT/EN/FR/DE/ES) e dichiarazione.
